css folder in assets digunakan sebagai template
animation : 
utk animation yg di gunakan saat ini baru hz atau hover zoom utk membesarkan sebanyak 20% dari asal saat di hover.

color : 
utk memberikan warna background dengan 1 gelap ke 9 terang. warna yg ada :
blue : bl
teal : tl
slate : sl
orange : or
red : rd


font : 
utk mengatur font : 
fs = font size (fs-s, fs-m, fs-l, fs-xl) utk ukuran font
af = align font (af-l, af-r, af-ctr) utk posisi text center secara vertical
fw = font weight (fw-lgt, fw-bld, fw-blk)
fc =font color (fc-w utk white,)
